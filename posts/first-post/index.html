My New Hugo Site

Hugo 사용법

  1. Install hugo
  2. Create github repo -> clone
  3. Create new site hugo new site 만들폴더이름
  4. Theme 설정
    1. git init
    2. git submodule add 테마깃헙주소 themes/테마이름
    3. echo ‘theme = “테마이름”’ » config.toml
  5. New post 작성
    • hugo new posts/포스트제목.md
  6. hugo server 열기 hugo server -D
