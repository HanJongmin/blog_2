### Hugo 사용법
1. Install hugo
1. Create github repo -> clone 
1. Create new site
    `hugo new site 만들폴더이름`
1. Theme 설정
    1. git init
    1. git submodule add 테마깃헙주소 themes/테마이름
    1. echo 'theme = "테마이름"' >> config.toml
1. New post 작성
    - hugo new posts/포스트제목.md
1. hugo server 열기
    ```hugo server -D```
